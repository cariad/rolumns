Rolumns
=======

**Rolumns** is a Python package for squishing data into rows and columns.

Rolumns supports:

* :doc:`Flat data <cook-book/flat>`
* :doc:`Grouped data <cook-book/grouping-lists>`
* :doc:`User-defined fields <cook-book/udfs>`

Installation
------------

.. code:: console

  pip install rolumns
